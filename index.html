Personal Projects

NYC Restaurant Health Inspection Letter Grade - Chrome Extension

Fetches a restaurant's health inspection letter grade (from NYC's Health Department) and displays the inspection card (as seen in the restaurant's front window) on Yelp, Zagat, Seamless, Menupages, Foursquare and Opentable.

Tech: JavaScript, Bluebird, CORS AJAX, SODA2 API, CSS

Instagram Infinite Scrolling - Chrome Extension

Enables infinite scrolling on Instagram's desktop site. Prior to a recent update, Instagram's desktop site would force a click each time the user wanted to load another page. They recently updated their site enabling infinite scrolling by default, however an initial click is still required, so this extension lives on.

Tech: JavaScript
